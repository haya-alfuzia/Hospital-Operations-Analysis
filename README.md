# Hospital Operations Performance Analysis

## Project Overview
This project focuses on analyzing hospital operational data using a realistic simulated dataset.
The objective is to evaluate performance, identify bottlenecks, and support data-driven decision making.

## Tools Used
- Power BI
- Microsoft Excel

## Key KPIs Analyzed
- Patient waiting time
- Appointment trends
- Department performance
- Resource utilization

## Analysis Process
- Data cleaning and preparation
- KPI definition
- Dashboard design and visualization
- Insight generation

## Outcome
An interactive Power BI dashboard was developed to provide clear insights into hospital operations
and highlight opportunities for performance improvement.
